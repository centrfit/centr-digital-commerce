# How to Apply

**No resume PDF. No cover letter. No recruiter screen.**

We built a different application process. You'll use Claude Code to generate a structured submission that combines your actual tool usage data with your professional background.

---

## Prerequisites

You need [Claude Code](https://docs.anthropic.com/en/docs/claude-code) installed. If you don't have it yet, it takes about 5 minutes to set up.

If you're not a Claude Code user yet — that's okay, but this role operates deep in it every day. It's how we build, analyze, and run operations. Consider trying it on real work for a week, then come back.

---

## Steps

### 1. Generate your insights report

If you haven't already, run `/insights` in any of your Claude Code projects. This generates a usage report showing what you build, how you use the tool, your biggest wins, and your usage patterns.

### 2. Clone this repo

```bash
git clone https://github.com/centrfit/centr-digital-commerce
cd centr-digital-commerce
```

### 3. Run `/apply`

Open this repo in Claude Code and run `/apply`. The command will:

- Read your Claude Code insights report
- Ask for your background info (current role, relevant experience, why this role)
- Generate a single `firstname-lastname.md` file combining everything

### 4. Review and submit

Review your generated file. Edit anything you'd like.

Upload it here: **[https://forms.gle/centr-digital-commerce](https://docs.google.com/forms/d/e/1FAIpQLSc_QVsZBLd3l4xT3yoRInpreZbCugIEcGM0jASvg8-YFnAsgQ/viewform)**

---

## What We're Looking For

Your insights report tells us how you actually work — not what you say on a resume. We're looking for:

- **Real usage** — operators who use Claude Code for real work, not people who installed it yesterday
- **Relevant domain** — building ecom systems, data pipelines, marketing ops, dashboards, analytics
- **Builder signal** — creating systems and infrastructure, not just asking questions

Your background section tells us if you have the right experience for this specific role.

Both matter. A great Claude Code user with no ecom experience won't fit. A great ecom operator who doesn't use tools this way won't fit either.
